# Instructions pour Claude Code

**Lis `AGENTS.md` à la racine du repo avant toute modification.** Il documente
l'architecture d'authentification non standard (Clerk détourné pour éviter sa
limite de mot de passe à 8 caractères), le système audio (pièges Safari
mobile, cache), les migrations de base de données, et les variables
d'environnement nécessaires. C'est le fichier de référence le plus important
de ce projet — la plupart des bugs déjà rencontrés et corrigés y sont
documentés pour éviter de les reproduire.

Voir aussi `docs/ROADMAP.md` pour les chantiers produit restants.
